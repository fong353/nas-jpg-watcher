# JPG DPI 修复逻辑

## 发现（是否需要修）

```
need_fix(jpg) = 
    jpg.JFIF.ResolutionUnit == "None"
    AND (
        jpg.Photoshop.XResolution > 1
        OR jpg.EXIF.XResolution   > 1
        OR jpg.JFIF.XResolution   > 1
    )
```

两条 AND，缺一即跳过。

## 修复

```
fix(jpg):
    jpg.JFIF.ResolutionUnit = "inches"

    # 优先级从低到高级联，后写覆盖前写
    if jpg.EXIF.XResolution exists:
        jpg.JFIF.XResolution = jpg.EXIF.XResolution
        jpg.JFIF.YResolution = jpg.EXIF.YResolution

    if jpg.Photoshop.XResolution exists:
        jpg.JFIF.XResolution = jpg.Photoshop.XResolution
        jpg.JFIF.YResolution = jpg.Photoshop.YResolution

    save(jpg, atomic=True, preserve_mtime=True)
    rename(jpg, "{basename}_fixed.{ext}")
```

## 段名映射（不同库可能命名不同）

| 段 | 别名 |
|---|---|
| JFIF | APP0 |
| EXIF | APP1 / IFD0 |
| Photoshop | 8BIM / APP13 / IRB |
